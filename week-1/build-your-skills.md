# Build Your Skills

These are starter prompts for building reusable Claude Code skills using the
skill-creator you installed during pre-work.

**How to use:**
1. Open Claude Code in your terminal
2. Type `/skill-creator` and press enter
3. When it asks what you want to build, paste one of the prompts below
4. Answer its follow-up questions for your specific market and workflow
5. The skill gets saved automatically -- run it anytime with `/skill-name`

---

## Competitive Monitor

Runs a full competitive landscape analysis on your market on demand.

```
I want to build a skill called competitive-monitor. It should run a full
competitive landscape analysis on my market using the context in my CLAUDE.md,
produce a feature comparison matrix and whitespace analysis, and save the
output as a dated markdown file in a research/ folder.
```

---

## Feedback Synthesizer

Extracts themes and product gaps from customer feedback, support tickets,
or survey data.

```
I want to build a skill called feedback-synthesizer. It should take customer
feedback, support tickets, or survey data I point it at or paste in, extract
the top themes and pain points, flag anything that looks like a product gap,
and save a structured markdown summary.
```

---

## Transcript Processor

Turns meeting and interview transcripts into structured, actionable summaries.

```
I want to build a skill called transcript-processor. It should take a meeting
or interview transcript file I point it at, extract key decisions, action
items, open questions, and themes, and save a structured summary as a markdown
file alongside the original transcript.
```
